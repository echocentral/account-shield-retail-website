And there's more

Account Shield gives you powerful capabilities to easily customize user content and journeys based on a wide range of characteristics:
  • perm_identity User
    User seen on site Boolean flag, to indicate whether that user has visited the site before.
    User seen on page Boolean flag, to indicate whether that user has visited the page before.
    User last seen Date when this user last accessed this site.
    User's devices Array of devices associated with the user.
    User's connections Array of connections associated with the user.
  • devices Device
    Device seen on site Boolean flag, to indicate whether that device has visited the site before.
    Device seen on page Boolean flag, to indicate whether that device has visited the page before.
    Device last seen Date when this device last accessed this site.
    Device type One of 'Mobile', 'Phablet', 'Tablet', 'Laptop' or 'Desktop'.
    Device metrics A host of properties describing the device, including: platform, browser, languages, screen dimensions, device orientation, etc.
  • wifi_tethering Connection
    IP Address Text string, providing the public IP address used by the connection.
    Connection seen on site Boolean flag, to indicate whether that connection has been used to visit the site before.
    Connection seen on page Boolean flag, to indicate whether that connection has been used to visit the page before.
    Connection last seen Date when this connection was last used to access this site.
    Connection Type One of 'Fixed', or 'Mobile'.
  • place Location
    Location co-ordinates Comma separated list of a latitude and longitude position of the city nearest to the location of the device.
    Country code A two character ISO code indicating the country where the device is located.
    Region A text string indicating the region of the country where the device is located.
    City A text string indicating the city nearest to the location of the device.
